# 🖼️ NFT Validator

> 🔍 **Verify NFT authenticity بسهولة using QR Code scanning**

This application allows you to:

* Enter a **collection address**
* Scan the NFT **QR code**
* Verify whether the NFT is authentic and belongs to the specified collection

---

## ⚙️ How It Works

1. Enter the **collection address**
2. Scan the NFT **QR code**
3. The app checks and verifies the NFT authenticity

---

## 🛠️ Framework Settings

| Setting          | Value           |
| ---------------- | --------------- |
| Root Directory   | `./`            |
| Build Command    | `npm run build` |
| Output Directory | `dist/public`   |

---

## 💡 Notes

* Make sure the **collection address is correct**.
* Verification depends on on-chain NFT data.
* Use trusted QR sources to avoid spoofed NFTs.

## Packages
html5-qrcode | Robust QR code scanning library with mobile rear camera support
framer-motion | Smooth page transitions and micro-interactions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge tailwind classes safely

## Notes
- App logic is entirely client-side as requested.
- Camera permissions will be requested from the user upon submitting the collection address.
- Uses `html5-qrcode` directly for better control over the scanning interface and lifecycle.
